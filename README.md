# Conference Paper Landing Page

This folder contains a simple static site that can be deployed with GitHub Pages.

## Files

- `index.html`: Temporary landing page for the paper.

## Before publishing

- Replace `Paper Title Goes Here` with the exact paper title in `index.html`.
- Update the release date text if needed.

## GitHub Pages

- Easiest path: put this folder's contents into a small public repository and enable GitHub Pages from the `main` branch.
- If your GitHub plan supports Pages from a private repository, the site itself is still public, so only publish non-sensitive text here.
